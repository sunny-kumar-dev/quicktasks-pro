# 🚀 QuickTasks Pro

A simple and modern Task Manager Web App built using HTML, CSS, and JavaScript.

🔗 Live Demo: https://quicktasks-pro.netlify.app/

---

## ✨ Features

- ✅ Add tasks
- ❌ Delete tasks
- 🎯 Clean and modern UI
- 📱 Responsive design (mobile-friendly)
- Dark & Light mode 

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
